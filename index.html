PARTSCOUNT

Computer Undergraduate Research Internship Science (CURIS) project under Prof. Leonidas J Guibas

Applied deep learning towards 3D Visual Question and Answering via training a deep neural network towards counting parts of a 3D object point-cloud. Scripts were implemented in Python and utilized PointNet and PartNet database for the dataset.

Reaching for the North

Final project for Introduction to Computer Graphics and Imaging

Utilized OpenGL, shaders, and raytracing to render the final image on Visual Studio. Models were created/simulated on 3DS Max.

Beveling the Reality

Final project for Interactive Computer Graphics

Implemented multiple functions ranging from modifying edges/vertices to spline interpolation (Hermite and Catmull-Rom) for a 3D modeling and animation program in C++ on Visual Studio. Modeled and animated a dragon and a shiba inu head to test its functionality.

Fwog Pot

Final project for Computer Graphics: Imaging Synthesis Techniques
